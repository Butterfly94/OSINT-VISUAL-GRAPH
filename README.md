# GraphLink — Визуализация графов и связей

Интерактивный инструмент для построения графов с поддержкой фото, описаний, кластеров и цепочек связей.

## Возможности

- 11 типов узлов (Персона, Организация, Локация, Событие, Устройство, Документ, Соцсети, Телефон, Автомобиль, Родственники, Компания)
- Загрузка фото и скриншотов к каждому узлу
- Создание связей между узлами с подписями
- Кластеры для группировки элементов
- Поиск, масштабирование, миникарта
- Импорт/экспорт в JSON

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub
2. Загрузи файлы:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USER/graphlink.git
git branch -M main
git push -u origin main
```
3. Перейди в **Settings → Pages → Branch: main → Save**
4. Сайт будет доступен: `https://YOUR_USER.github.io/graphlink/`

## Деплой на Railway

1. Загрузи репозиторий на GitHub
2. Зайди на [railway.app](https://railway.app)
3. **New Project → Deploy from GitHub repo**
4. Выбери репозиторий — Railway автоматически задеплоит
5. **Settings → Generate Domain** для получения публичного URL

## Управление

| Действие | Как |
|---|---|
| Добавить узел | Двойной клик на холст или кнопка «＋ Узел» |
| Создать связь | Навести на узел → потянуть за точку на краю |
| Редактировать | Двойной клик на узел |
| Удалить | Выделить + Delete |
| Масштаб | Колесо мыши |
| Перемещение | Зажать и тянуть холст |
| Поиск | Ctrl + F |
